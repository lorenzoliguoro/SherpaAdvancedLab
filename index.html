Synthetic reading

Methodological note. This tool supports qualitative and semi-quantitative causal exploration. The Input Matrix helps identify variables associated with a phenomenon. The Causal Matrix translates them into explicit causal relationships. The System Map visualises the emerging causal structure and identifies possible feedback loops. The Shock Analysis uses polarity, qualitative intensity and time responsiveness to estimate plausible transmission pathways. The output is not an econometric forecast. It does not calculate real elasticities, substitution rates or investment response functions. It supports structured discussion, scenario exploration and collective sense-making.
